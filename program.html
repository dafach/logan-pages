Shoulder Maintenance (Every Session, 5 min)

Do these before your first working set:

  • Band Pull-Aparts — 2x20 (light band, warm-up)
  • Cable External Rotation — 2x15/side (Ancore, light weight)
  • Dead Hangs — 2x20-30 sec (pull-up bar)

Non-negotiable while the shoulder is healing. 5 minutes. Protects everything else in the session.

Progression Rules

System▶

Rep Range Double

  • Hit top of rep range on Set 1 → increase weight next session
  • Other sets: keep same weight as long as you get more than half the target reps
  • If you can't get half the reps on a later set, reduce weight by one increment

Dynamic Double

  • Hit top of rep range on Set 1 → increase weight next session
  • If Set 1 stalls (same reps two sessions in a row) → drop 5%, rebuild
  • Later sets naturally trail behind — that's fine

Rest Periods

  • Compound lifts: 90 seconds
  • Isolation/accessories: 60 seconds
  • Keep it honest — this is how you stay under 50 min

Why This Program

Rationale▶
  • 3x/week full body — miss a day, every muscle still gets hit 2x that week
  • Push/Pull/Legs every session — balanced, no anterior dominance
  • 14-15 sets per session — fits 45-50 min with proper rest periods
  • Mario's progression system — clear rules for when to add weight, when to deload
  • Posterior chain covered — chin-ups, rows, face pulls, RDLs, leg curls, reverse flyes
  • Shoulder-friendly — neutral grip OHP, face pulls every Session A, prehab built in
  • Equipment-matched — everything uses your PRX, Ancore, DBs, machines, and dip/pull-up station
